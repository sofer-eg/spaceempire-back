# Ship class catalog (ct_ship_classes)

Статичный справочник классов кораблей, портированный из оригинального
StarWind (`sql/db.sql`, таблица `ct_ship_classes`, 86 строк). Аналог
справочника товаров (`balance.Goods`): данные в YAML, в коде — immutable
каталог, в БД ничего.

## Источник

- Оригинал: `starwind/sql/db.sql` → `INSERT INTO ct_ship_classes`.
- Конвертер: `cmd/starwind-tools/convert-ship-classes` читает дамп и пишет
  `configs/ship_classes.yaml`. Дамп фактически в UTF-8 (несмотря на
  `DEFAULT CHARSET=cp1251`), читается напрямую; iconv не нужен.
- Перегенерация: `go run ./cmd/starwind-tools/convert-ship-classes -sql
  <db.sql> -out configs/ship_classes.yaml`.

## Колонки (из ct_ship_classes)

| поле | смысл |
|------|-------|
| `id` | PK строки каталога |
| `race` | раса-владелец: 1–8, `100` = уникальные/служебные |
| `type` | номер слота в пределах расы (1–9 стандарт, 10+ доп./уникальные) |
| `class` | игровой класс → категория (см. ниже) |
| `name` | имя корабля («Колосс», «Меркурий», …) |
| `speed`, `acceleration`, `maneuverability` | кинематика (per-tick StarWind) |
| `laser` | бюджет лазерной мощности |
| `shield`, `hull`, `shield_charge` | щит/HP/зарядка |
| `cargobay` | грузовместимость |
| `base_price` | базовая цена на верфи |
| `hanger_small`, `hanger_capital` | вместимость ангаров (мелкие/крупные) |
| `hanger_ship_type` | какой слот сам занимает (1 крупный, 2 мелкий) |
| `hanger_ship_space` | сколько места занимает в ангаре |
| `pilot_cabin` | сколько пилотов влезает |
| `jump_fuel` | топливо на прыжок |

## Категории (маппинг колонки `class`)

X-universe-таксономия. Уникальные корабли переиспользуют базовый `class`,
наследуя категорию.

| class | code | label |
|-------|------|-------|
| 1 | M1 | Носитель |
| 2 | M2 | Эсминец |
| 3 | M3 | Тяжёлый истребитель |
| 4 | M4 | Истребитель |
| 5 | M5 | Разведчик |
| 6 | M6 | Корвет |
| 7 | TL | Супертранспорт |
| 8 | XX | Специальный (флагман/служебный слот) |
| 9 | TS | Транспорт |

Неизвестный `class` → `CategorySpecial` (без паники).

## API в коде (`internal/balance`)

- `ShipClass` — одна строка каталога (typed `domain.ShipClassID`).
- `(ShipClass).Category() ShipClassCategory`, `CategoryLabel(cat)`.
- `ShipClasses` — immutable каталог: `GetShipClass(id)`, `AllShipClasses()`,
  `ShipClassCount()`, `ShipClassesByRace(race)`.
- `LoadShipClassesFromFile(path)` — загрузка YAML с валидацией
  (id>0, name!="", без дублей id).

## HTTP

- `GET /api/ship-classes` (open, без auth) → `{items: [...]}`. Каждый item:
  `id, race, type, class, category, categoryLabel, name, speed,
  acceleration, laser, shield, hull, cargobay, basePrice, pilotCabin`.
- Фронт: `fetchShipClasses()` в `front/src/api.ts`.
